#### 📌 기본적인 반복문 비교

| 메서드       | 대상             | 중단 가능?                             | 반환값              | 주요 특징                                   |
|--------------|-------------------|----------------------------------------|---------------------|---------------------------------------------|
| `for`        | 배열 & 유사배열   | ✅ 가능 (`break`, `continue`,`return`) | ❌ 없음             | **최대 유연성, 속도 빠름**                 |
| `forEach()`  | 배열만            | ❌ 불가능                              | ❌ 없음             | `break` / `return false` 안 됨             |
| `every()`    | 배열              | ✅ 가능 (`false 만나면 중단`)          | **`true` / `false`** | **모든 요소가 조건 만족하면 `true` 반환** |
| `some()`     | 배열              | ✅ 가능 (`true 만나면 중단`)           | **`true` / `false`** | **하나라도 만족하면 `true` 반환**         |
| `each()`     | jQuery 객체 & 배열| ✅ 가능 (`return false`)               | ❌ 없음             | jQuery에서 사용 가능                        |

<br>

#### 📌 언제 어떤 반복문 사용?

###### 1️⃣ 반복을 유연하게 제어 (break, continue) 해야 한다면? → for 문 사용!

###### 2️⃣ 중단 없이 간단한 반복만 필요하다면? → forEach() 사용!

###### 3️⃣ jQuery 객체 포함 순회해야 한다면? → each() 사용!

###### 4️⃣ 모든 요소가 조건을 만족하는지 검사하려면? → every() 사용!

###### 5️⃣ 하나라도 만족하는 요소가 있는지 검사하려면? → some() 사용!

<br>

#### 예제
```javascript

var arr = [1, 2, 3, 4, 5];
var result = [];

// 1.for문 (최대 자유도)
for (let i = 0; i < arr.length; i++) {
    if (arr[i] % 2 === 0) {
        result.push(arr[i] * 10);
    }
}

// 2.forEach() (중단 불가)
arr.forEach(num => {
    if (num % 2 === 0) {
        result.push(num * 10);
    }
});

// 3.every() (모두 만족해야 true)
// false 만나면 루프 중단
let isAllEven = arr.every(num => num % 2 === 0);
console.log(isAllEven);

// 4.some() (하나라도 만족하면 true)
// true 만나면 즉시 중단
let hasEven = arr.some(num => num % 2 === 0);
console.log(hasEven);

// 5.reduce() (누적값 계산)
// 배열 값을 누적 계산할 때 사용
let sum = arr.reduce((acc, num) => acc + num, 0);
console.log(sum);
```

#### 반복문 & 콜백 함수 

> 콜백 함수 :  다른 함수의 인자로 전달되어 실행되는 함수

> 콜백 함수는 반복문의 동작을 결정하는 핵심 역할

> forEach, map, filter, some, every(배열 메서드), 콜백 함수를 매개변수로 받아 배열의 각 요소에 대해 특정 작업 수행

> 대부분의 배열 메서드의 콜백 함수는 currentValue, index, array를 기본파라미터로 받음 (reduce는 accumulator 까지) 

<br>

####  반복문 & 콜백 함수 비교  

| 종류         | 콜백 함수           | break             | return             | 콜백 함수 목적                 | 새로운 배열 생성 |
|--------------|---------------------|--------------------|-------------------|---------------------------------|-------------------|
| `for`        | ❌                  | ✅                | ✅                | 일반 루프                      | ❌                |
| `forEach()`  | ✅                  | ❌ (중단 불가)    | ❌                | 단순 반복 (중단 X)             | ❌                |
| `map()`      | ✅                  | ❌ (중단 불가)    | ✅                | 새로운 배열 생성               | ✅                |
| `filter()`   | ✅                  | ❌ (중단 불가)    | ✅                | 조건을 만족하는 요소 필터링     | ✅                |
| `some()`     | ✅                  | ✅                | ✅                | 조건 만족 시 true 반환 후 중단  | ❌               |
| `every()`    | ✅                  | ✅                | ✅                | 모든 요소가 조건 만족해야 true  | ❌               |
